# CtfWatcherBot
CtfWatcherBot is a Telegram Bot that helps you keep track of Capture The Flag competitions using [CTFTime Api](http://ctftime.org/api/). <br>
Try yourself at [https://t.me/CtfWatcherBot](https://t.me/CtfWatcherBot)
